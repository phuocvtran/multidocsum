# Multidocsum
Multi-document summarization for Vietnamese news.

# Commands
```shell
# Clone this repo
git clone https://github.com/phuocvtran/multidocsum.git
# Install multidocsum and all its dependencies
cd multidocsum
pip install .
# Download & preprocess ViMs dataset
python setup_data.py
```
